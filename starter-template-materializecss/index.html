YACS 2.0 - The Better Course Scheduler

YACS 2.0 allows for student input and review on individual classes in order to assist peers with a seemless transition between semesters. Typically, when a student drops a class it is because the class is not what he or she expected. YACS 2.0 aims to reduce the number of classes dropped per semester through a new student feedback system. With feedback on both the professor and the course, students will be better advised which courses to take and with which professor.
